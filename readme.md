The Stock Management provides the data for Global Beverage Corporation Exchange. 

This system handles basic stock market operations:
- Calculates dividend yield and P/E ratios for stocks
- Records trades with timestamps
- Computes volume weighted stock prices
- Calculates the GBCE All Share Index using geometric mean

This will demonstrate all the functionality with sample data from the GBCE.

## Sample stocks included

- TEA (Common stock)
- POP (Common stock)
- ALE (Common stock)
- GIN (Preferred stock)
- JOE (Common stock)

To Run the Script , just trigger the main.py file.
